# 工作内容:

全部的工作就是分为标注视频和根据关键词爬取文本

关键词就是上次发在学姐发在群里的那个,

爬取文本的代码就是code文件夹中的

爬取的平台需要有三个,抖音,快手,哔哩哔哩

代码有些平台不能直接用,需要使用ai稍微改一下,因为平台也在不断地改进反爬机制

代码中有些许要改的点:先看一遍code文件夹下的readme文件,里面有绝大多数的说明,

配置环境部分有些麻烦,建议大家安装一下anaconda,之后给这个项目单独创建一个环境,在这个环境之中下载相应的包和库

# 分工:

现在的设想是每个人都试着搞一下,如果最后都成功的话,那之后的工作就是每周每个人上传爬取到的文本和标注的数据,数据的话每个人每周210个就行,但210是最低下限,因为老师那边赶的比较紧

当然上面是最好的情况(每个人都成功了),不过至少需要3个人来爬取这个文本,因为平台不一样,抖音,快手,哔站,最后按照两人一组的情况进行搭配,响应的爬取文本的同学负责标注的50条数据就需要同组的另一位同学承担,也可以组内协商,自由定夺

初步是每个人尝试两个平台(任选),在下星期三晚上之前,我会汇总一下,最后确定最后的分工工作

第三周的话就是总量大概在1000左右,每个人分到的就是200条左右,其余的任务就是把这个搞一下,最后实现我们自己爬取文本
